This app compiles with the AOT compiler!

You also need to instruct the AOT compiler to use your translation configuration. To do so, you configure the translation with three options in your angular.json file.

When you internationalize with the AOT compiler, you must pre-build a separate application package for each language and serve the appropriate package based on either server-side language detection or url parameters. ?? How??

i18nFile: the path to the translation file.

i18nFormat: the format of the translation file.

i18nLocale: the locale id.
